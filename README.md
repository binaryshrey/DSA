### 2. Hello World
```Java

import java.lang.*;
import java.io.*;

/* Name of the class has to be "Main" only if the class is public. */
class HelloWorld
{
	public static void main (String[] args) throws java.lang.Exception
	{
	    System.out.println("Hello World");
	}
}
```
